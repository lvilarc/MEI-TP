Input: s = "(()"
Output: 2
Explanation: The longest valid parentheses substring is "()".
Example 2:

Input: s = ")()())"

Explanation: The longest valid parentheses substring is "()()".
Example 3:

Input: s = ""
Output: 0

0 <= s.length <= 3 * 10^4
