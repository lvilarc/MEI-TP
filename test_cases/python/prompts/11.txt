Generate unit tests in Python for the function that solves the problem described below.

Requirements:
- Produce exactly 5 test cases covering normal cases, edge cases, and boundary inputs.
- Use the standard LeetCode Python method name for this problem.
- Do NOT implement the solution function. Assume the implementation is provided separately in a module passed as solution_module.
- Output a single self-contained Python file that defines run_tests(solution_module) and verifies each result using assert statements.
- Return only the Python code, with no explanation.

Problem:

Minimum Window Substring

Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character in t (including duplicates) is included in the window. If there is no such substring, return the empty string "".

The testcases will be generated such that the answer is unique.

 

Example 1:

Input: s = "ADOBECODEBANC", t = "ABC"
Output: "BANC"
Explanation: The minimum window substring "BANC" includes 'A', 'B', and 'C' from string t.
Example 2:

Input: s = "a", t = "a"
Output: "a"
Explanation: The entire string s is the minimum window.
Example 3:

Input: s = "a", t = "aa"
Output: ""
Explanation: Both 'a's from t must be included in the window.
Since the largest window of s only has one 'a', return empty string.
 

Constraints:

m == s.length
n == t.length
1 <= m, n <= 10^5
s and t consist of uppercase and lowercase English letters.
